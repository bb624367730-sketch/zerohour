## Phase 2：举办 Testnet 锦标赛（Day 4-7，周四~周日）

详见 `TESTNET_TOURNAMENT.md`，核心执行要点：

### 每日固定动作

| 时间 | 动作 |
|------|------|
| 10:00 | 发昨日战报（总交易量、完成轮数、最大单笔、活跃玩家数） |
| 14:00 | 发一条有趣数据（"目前 Pepe 战队领先，但猫猫队人均分红最高"） |
| 20:00 | 发当前榜单 + 倒计时提醒 |
| 任意 | Telegram 群实时互动，有人大额买票立即在群里喊 |

### 战队士气战

每天在 Twitter 发起投票：**"今天谁是最强战队？"**

- 制造战队之间的对抗感
- 投票本身就是传播
- 优胜战队全员额外奖励

### 数据追踪（用于 grant 申请）

每天记录：
- 独立钱包地址数
- 总交易笔数
- 总交易量（testnet SUI）
- 完成轮数
- 各战队数据（人数、票数、交易量）
- Telegram 群人数增长
- Twitter 互动数据

> ZeroHour 是 Sui 上的 Fomo3D 风格链游 — 选战队、买票、实时分红、倒计时归零赢奖池。
